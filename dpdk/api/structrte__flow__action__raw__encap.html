RTE_FLOW_ACTION_TYPE_RAW_ENCAP

Raw tunnel end-point encapsulation data definition.

The data holds the headers definitions to be applied on the packet. The data must start with ETH header up to the tunnel item header itself. When used right after RAW_DECAP (for decapsulating L3 tunnel type for example MPLSoGRE) the data will just hold layer 2 header.

The preserve parameter holds which bits in the packet the PMD is not allowed to change, this parameter can also be NULL and then the PMD is allowed to update any field.

size holds the number of bytes in data and preserve.
