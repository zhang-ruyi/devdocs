Detailed Description

RTE_FLOW_ITEM_TYPE_E_TAG.

Matches a E-tag header.

The corresponding standard outer EtherType (TPID) value is RTE_ETHER_TYPE_ETAG. It can be overridden by the preceding pattern item.
