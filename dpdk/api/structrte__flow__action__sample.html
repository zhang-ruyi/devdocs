RTE_FLOW_ACTION_TYPE_SAMPLE

Adds a sample action to a matched flow.

The matching packets will be duplicated with specified ratio and applied with own set of actions with a fate action, the sampled packet could be redirected to queue or port. All the packets continue processing on the default flow path.

When the sample ratio is set to 1 then the packets will be 100% mirrored. Additional action list be supported to add for sampled or mirrored packets.
